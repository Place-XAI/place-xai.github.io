FlowX — AI Visual Notebook

A node-based visual environment for building, running, and explaining GeoAI workflows — no code required. Drag, connect, and inspect every step.

Related papers: Elevating the RRE framework for geospatial analysis with visual programming platforms — IJAEOG, 2024; Visual programming-based geospatial cyberinfrastructure for open-source GIS education 3.0 — CaGIS, 2025.

🧩

Visual, node-based

Compose spatial pipelines by connecting nodes — readers, transforms, models, and views.

🔍

Explainable by design

Inspect intermediate results and model explanations at every step of the flow.

🗺️

Geospatial-native

Built-in support for vector, raster, and network data and maps.
