Network of evolvable neural units can learn synaptic learning rules and spiking dynamics

Paul Bertens · Seong-Whan Lee Published 2020-12-10 DOI 10.1038/s42256-020-00267-x Source nature.com ↗
Core Conclusion

An Evolvable Neural Unit (ENU) — a gated recurrent network placed at every synapse and neuron — can independently evolve spiking dynamics and reinforcement-type learning rules without any predefined mathematical model. A network of ENUs, optimized via evolution strategies, discovers integrate-and-fire behaviour, spike-timing-dependent plasticity, and one-shot avoidance learning in a T-maze task — opening a path toward biologically realistic AI that learns without backpropagation and generalizes from a single reward experience.

★

Why It Matters

  • Frees neural computation from hardcoded rules. Prior spiking network models evolve hyper-parameters of a fixed predefined model; ENUs evolve the entire function of each synapse and neuron — letting biology emerge rather than being assumed.
  • Parameter-efficient by design. Because all ENU compartments share one weight matrix, total evolvable parameters (~12,000) remain fixed regardless of network size — a rare scalability property in neural architecture search.
  • One-shot learning without backpropagation. The evolved ENU-NN learns from a single negative experience (eating poison once) and adapts immediately — a capability conventional deep networks still struggle with.
  • Bridges computational neuroscience and AI. ENUs independently rediscover IAF spiking and neuromodulated STDP, validating that complex biological learning rules can emerge from generic optimization rather than domain expertise.
  • Open and reproducible. All code is publicly available via a Code Ocean compute capsule (10.24433/CO.1361267.v1); no external datasets are required — the entire paper is reproducible from source.

Core Methods

  • Evolvable Neural Unit (ENU): a modified GRU with an extra feedback output gate. Each gate is a single-layer network with evolvable weight matrix W. The memory state stores dynamic parameters (analogous to membrane potential, synaptic weight); the output gate enables spike-threshold behaviour via clipping.
  • Weight sharing across all compartments: all synapse ENUs share one weight matrix; all neuron ENUs share another. Only ~12,000 parameters are evolved regardless of network size — critical for scalability.
  • Evolution strategies (ES): gradient approximated by Gaussian sampling over parameter space (Algorithm 1); fitness ranking (top ~20% of offspring account for 80% of gradient, α = 5); mini-batch environments for fitness smoothing; momentum SGD on the ES gradient.
  • Network construction: synapse ENUs receive pre- and postsynaptic neuron inputs (the postsynaptic feedback channel enables STDP-like rules); synapse outputs are summed and processed by neuron ENUs; connection matrix is sparse for the T-maze task.
  • Task-specific fitness functions: IAF — minimize interspike-interval error + spike-intensity error; STDP — minimize MSE against the STDP weight-update function; T-maze RL — maximize cumulative reward with energy-decay exploration term.
  • Ablation study: compares ENU vs. GRU, LSTM, NFENU (no feedback), SHAREDENU (shared neuron/synapse weights) across all tasks; statistical significance validated over 30 trial runs (p < 0.005).

① Map the evolved ENU weights onto resistive-memory in-memory computing hardware

Extending this paperThe ENU-NN discovers spiking dynamics in software running on GPU tensors, but remains far from neuromorphic hardware. A direct extension: map the evolved ENU weight matrices onto resistive memory crossbar arrays where analog conductance states store the ENU's dynamic parameters (membrane potential analogs, synaptic weights). The paper's weight-sharing property (~12,000 parameters total) makes this particularly feasible — each crossbar tile implements one shared ENU gate, while individual cells encode the unique internal state per compartment. This moves the paper's biologically inspired claim from software analogy to physical substrate, closing the loop from evolved software to neuromorphic silicon.

Added value to my researchFor spatiotemporal GeoAI, neuromorphic in-memory architectures offer a low-power edge-inference path — for example, embedding graph convolution in IoT health-monitoring sensor nodes or GPS trajectory chips. The ENU's parameter-sharing structure directly maps to the weight-tied graph convolution kernels I use in physics-informed graph learning, pointing toward the same design principle for reducing memory footprint in deployment at the urban-sensor edge.

② Extend the T-maze to a road-network graph and evolve graph-aware spatial routing policies

Extending this paperThe paper's T-maze is a minimal 5×4 grid with two choice points. Extending it to an arbitrary road-network graph (nodes = intersections, edges = street segments with travel-time weights) would force the ENU-NN to evolve not just memory for one food/poison location but a generalizable spatial navigation policy across varied topologies. The paper already shows that ENUs rediscover inhibitory synapse behaviour after one negative reward — that same one-shot inhibition mechanism, replicated across a richer graph topology, seeds a graph-embedded reinforcement agent that internalizes network structure through evolution rather than explicit GNN message passing.

Added value to my researchThis directly intersects my physics-informed graph learning agenda: an ENU-NN that evolves routing on a street graph is a testbed for whether biologically grounded local plasticity rules can replace global backpropagation in GNN training — especially relevant for sparse, irregular spatial graphs (transit networks, 2SFCA catchment graphs) where standard GNN training is numerically unstable.

Cross-ref (corpus): Discovering cognitive strategies with tiny recurrent neural networks — 10.1038/s41586-025-09142-4

③ Test whether the evolved ENU learning rule avoids catastrophic forgetting in a continual task stream

Extending this paperThe paper resets all ENU internal states every generation — each agent is a blank slate that must relearn within an episode. But this intra-episode plasticity is precisely what makes ENUs interesting for continual learning: the evolved gate weights are fixed post-evolution and only internal states adapt, so the rules could in principle generalize across tasks without forgetting. A direct extension: expose the ENU-NN to a sequential stream of diverse environments and measure whether the evolved rule shows zero-shot generalization. The current paper never tests inter-task transfer, leaving the strongest claim about generality unverified.

Added value to my researchContinual learning of spatial context is an open problem in GeoAI: a model trained on one city's health-facility access patterns should not forget when fine-tuned on another city. An evolved meta-learning rule (ENU-style) that updates local states without modifying shared weights is a principled alternative to rehearsal-based or regularization-based continual learning for my multi-city spatial accessibility models.

Cross-ref (corpus): Continual learning of context-dependent processing in neural networks — 10.1038/s42256-019-0080-x

④ Enrich the reward channel with multi-timescale neuromodulatory signals

Extending this paperThe paper's reward neuron uses one channel for positive and one channel for negative reward — a coarse two-bit neuromodulation signal. Neuroscience evidence shows that dopamine operates on multiple timescales simultaneously (tonic vs. phasic, fast vs. slow decay). Extending the ENU's reward input to a vector of neuromodulatory channels, each with a different temporal decay constant (themselves evolvable), would let the ENU-NN discover which timescale combinations yield the fastest one-shot learning — turning the paper's single reward channel into a structured neuromodulatory bus.

Added value to my researchSpatiotemporal kernel density estimation in my work relies on fixed bandwidth parameters per temporal scale (daily, weekly, seasonal). Evolved multi-timescale neuromodulatory channels are architecturally analogous to learned bandwidth parameters per temporal resolution — the ENU framing suggests these bandwidths could be jointly learned rather than tuned by cross-validation, applicable to GPS trajectory and mobile-phone record analysis.

Cross-ref (corpus): Multi-timescale reinforcement learning in the brain — 10.1038/s41586-025-08929-9

⑤ Apply the weight-sharing principle to build a parameter-efficient spatiotemporal GNN for health geography

Extending this paperThe ENU's central design insight — shared gate weights with unique per-compartment internal states — solves a parameter-explosion problem: a network of thousands of compartments needs only ~12,000 evolvable parameters because computation is local but rules are global. This principle could extend the ENU-NN beyond toy mazes: replace the sparse fixed topology with a dynamic population graph (nodes = residential zones, edges = spatial adjacency or travel-time links), where synapse ENUs serve as attention-based edge message functions and neuron ENUs serve as node aggregators. Evolving this on a population health outcome task would test whether local Hebbian-like rules can substitute for global gradient descent in large spatial graphs.

Added value to my researchMy 2SFCA/R2SFCA accessibility models compute supply-demand ratios at every node of a spatial network — structurally identical to the ENU's per-synapse local computation. A weight-shared, state-per-node ENU-GNN could learn an accessibility update rule from mobility data, auto-discovering the distance-decay function and competition radius — replacing both the Gaussian kernel assumption and the fixed catchment radius with evolved, data-driven local rules. This is a concrete step toward a GeoAI foundation model for accessibility that transfers across city types without retraining from scratch.
