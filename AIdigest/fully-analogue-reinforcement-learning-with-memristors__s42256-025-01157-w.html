Fully analogue reinforcement learning with memristors

Yue Zhang · Xiaojuan Qi · Zhongrui Wang Published 2025-12-09 DOI 10.1038/s42256-025-01157-w Source nature.com ↗
Core Conclusion

Portner et al. demonstrate the first fully analogue, on-chip reinforcement learning system: an actor–critic network implemented entirely in hafnium-oxide memristor crossbar arrays performs storage, inference, error computation, and weight updates without any digital multiply–accumulate units or external memory traffic. An analogue temporal-difference (TD) voltage is the sole learning driver; the system solves T-maze navigation with only 27 physical devices and a continuous Morris water-maze task with approximately 1,000 synapses, pointing toward autonomous edge devices that literally learn inside their own hardware.

★

Why It Matters

  • Closes the digital gap in neuromorphic RL. Prior memristor-based RL always offloaded training to conventional digital CPUs; this work is the first to keep every computation — including the TD error and weight update — inside the analogue hardware, eliminating the memory-wall bottleneck.
  • Ohm's and Kirchhoff's laws do the algebra. Weighted summation is performed physically via current accumulation across the crossbar; no data travel outside the array, and power is consumed only where conductance changes occur.
  • Self-correcting noise tolerance. Because the TD error is recomputed at each step from the new (possibly imperfect) conductance, residual stochastic programming errors are automatically fed back with opposite sign, making the system robust to device variability.
  • Proof-of-principle at neuroscience benchmarks. The T-maze and Morris water-maze tasks are classical behavioural neuroscience assays, bridging hardware engineering directly to biological learning theory.
  • Pathway to sustainable edge AI. By eliminating global gradient broadcast and external memory traffic, the approach could enable autonomous drones, biomedical implants, and exploratory robots to learn from a small battery for months.

Core Methods

  • In-memory computation via crossbar arrays: memristor conductances store weight matrices; applying row voltages produces column currents representing weighted sums via Ohm's + Kirchhoff's laws — no external memory transfer.
  • Actor–critic architecture on hardware: two sets of physical conductances represent actor (policy) weights and critic (value-function) weights, mapped onto the same crossbar substrate.
  • Analogue temporal-difference (TD) error generation: a voltage-divider sub-circuit (two critic memristors + fixed reference) computes the scalar TD error as an analogue voltage at each timestep.
  • Pulse-based weight update: the TD error voltage is translated into a number of programming pulses that directly adjust target conductances — no digital gradient computation required.
  • Self-correcting noise handling: stochastic programming errors are automatically compensated because the TD error is recomputed from the current (imperfect) conductance state at every step, feeding back residual error with opposite sign.
  • Calibrated software twin: for the T-maze experiment, a portion of connections is handled by a software model calibrated to physical device characteristics — a hybrid step toward fully physical operation flagged as a remaining limitation.

① Scale the analogue TD rule from crossbar to physical graph neural networks for spatial optimization

Extending this paperThe paper's analogue TD loop operates on a flat weight matrix (crossbar) and is validated on simple navigation benchmarks (T-maze, water-maze). A direct extension: map the actor–critic architecture onto a physical graph neural network substrate where edges correspond to memristor pairs and nodes to spatial locations — enabling the TD signal to propagate along graph topology rather than a dense crossbar. This would let the analogue hardware learn directly on graph-structured state spaces (road networks, transit graphs) without digital preprocessing. The corpus work on resistive-memory liquid-state machines already encodes temporal spatio-structural information in RRAM; combining both could yield a hardware-native graph RL solver.

If the analogue TD rule propagates natively on a memristor graph, it becomes possible to run on-chip facility-location optimization — where each node is a candidate service site and the critic evaluates accessibility scores directly in analogue hardware, bypassing the digital simulation bottleneck in my 2SFCA scaling work.

② Replace the analogue TD error with an in-memory Bayesian uncertainty signal for active spatial data collection

Extending this paperThe paper's learning signal is a scalar TD voltage. A meaningful extension: replace or augment it with a Bayesian uncertainty estimate computed in analogue hardware — leveraging the inherent stochasticity of memristor conductance variations as a natural Monte Carlo ensemble rather than noise to suppress. Deep Bayesian active learning using in-memory computing hardware (doi:10.1038/s43588-024-00744-y) already shows that in-memory hardware can output calibrated uncertainty; combining that with the analogue TD loop would enable an RL agent that explores more in regions of high predictive uncertainty and less where the actor–critic is confident.

Added value to my researchIn my spatial data-collection work (mobile phone, GPS trajectories, social media), the key question is where to deploy sensors or agents next. A hardware-native Bayesian RL agent could drive active spatiotemporal sampling — selecting next measurement locations by trading off uncertainty reduction against travel cost, directly paralleling 2SFCA service-area delineation where we must decide which zones need more data.

③ Import physical backpropagation advances to eliminate the remaining digital dependency

Extending this paperThe article explicitly flags its key limitation: some architectural modules still depend on digital circuitry (the calibrated software twin). Training of physical neural networks (doi:10.1038/s41586-025-09384-2) develops a general framework for training analogue physical systems without digital gradient flow, showing that complex networks can learn through local physical perturbations alone. Extending the memristor RL paper: substitute the calibrated software twin with perturbation-based physical training, achieving a truly zero-digital actor–critic loop. This directly addresses the limitation the authors flag and could allow the system to scale to the ~1,000-synapse water-maze regime entirely on physical hardware without any digital modules.

Added value to my researchMy agent-based modeling work (Monte Carlo simulations of urban mobility) currently runs entirely on CPU clusters. If physical training can replace digital gradient computation, it opens a path to hardware-accelerated agent-based spatial simulation — memristive crossbars as massively parallel Monte Carlo engines for spatiotemporal dynamics modeling.

④ Transfer the analogue actor–critic principle to CMOS-compatible synaptic transistors for scalable urban RL

Extending this paperHfOx memristors are not yet manufacturable at standard CMOS scale. Synaptic and neural behaviours in a standard silicon transistor (doi:10.1038/s41586-025-08742-4) demonstrates that standard silicon transistors can exhibit synaptic plasticity (long-term potentiation/depression) without exotic materials. Extending the memristor RL paper: reformulate the voltage-divider TD circuit using CMOS-compatible synaptic transistors, retaining the self-correcting analogue update rule on a platform compatible with existing foundry processes. This would lower the barrier to deploying analogue RL in real-world large-scale systems dramatically.

⑤ Leverage reward-magnitude insights to calibrate analogue TD voltage for geospatial reward design

Extending this paperThe paper treats reward as a given scalar and does not study how reward magnitude interacts with analogue TD computation. Reward magnitude determines reinforcement learning efficiency (doi:10.1126/science.aeb0813) shows that reward scale systematically governs learning speed and efficiency across biological and artificial agents. Extending the memristor RL paper: systematically vary the voltage amplitude of the reward signal and measure its effect on convergence speed and memristor programming noise — translating the reward-magnitude principle into an analogue voltage-scale calibration protocol that optimizes hardware learning efficiency without changing device architecture.

Added value to my researchIn my spatial RL applications (agent-based models for healthcare access or EV routing), reward shaping is typically done by trial and error. If reward magnitude has a principled effect on analogue hardware convergence, it provides a physics-grounded reward-design heuristic for my spatiotemporal agent models — linking the scale of a spatial utility function (e.g., accessibility improvement per episode) to hardware learning rate without hyperparameter search.
