Stable learning establishes some common ground between causal inference and machine learning

Cui, Peng · Athey, Susan Published 2022-02-23 DOI 10.1038/s42256-022-00445-z Source nature.com ↗
Core Conclusion

Spurious correlation — arising from confounding, data selection bias, or non-causal associations — is the root cause of the instability, unexplainability, and unfairness of modern machine learning models. Stable learning, a framework bridging causal inference and ML, addresses this by using sample reweighting to decorrelate input features, allowing predictive models to identify direct causal variables and achieve uniformly good performance across unknown distribution shifts — without requiring the strict assumptions of full causal identification. This Perspective argues that causality can improve ML's existing stronghold (prediction) itself, not merely extend it to intervention and decision-making.

Why It Matters

  • Reframes the ML-causality relationship. Most discourse treats causal inference as a separate paradigm for intervention; this paper argues causality can directly improve predictive stability, making both fields mutually beneficial rather than competing.
  • Identifies a unified root cause. Stability, explainability, and fairness failures in ML share a common mechanism — spurious correlation from confounding and selection bias — and stable learning attacks all three simultaneously.
  • Loosens the strict assumptions of causal inference. By framing the goal as stable prediction rather than consistent causal effect estimation, stable learning can make progress even when full unconfoundedness, overlap, or SUTVA assumptions cannot be satisfied.
  • Actionable methodology. The paper links theory to practical algorithms (SRDO, deep global balancing) that can be plugged into standard learning pipelines as regularizers, lowering the barrier for practitioners in high-stakes domains.
  • Addresses high-stakes deployment risks. In healthcare, criminal justice, and finance, where distribution shifts are inevitable and errors have outsized social consequences, stable prediction is not optional — this paper provides a principled path forward.

Core Methods

  • Spurious correlation taxonomy: classifies three mechanisms generating correlations — causal, confounding, and selection bias — and shows why only causal correlations are stable across distribution shifts (Fig. 1).
  • Sample reweighting for global balancing: the central technical contribution; for each feature treated iteratively as a "treatment," sample weights are learned to decorrelate it from remaining features (covariates), removing spurious correlations globally.
  • Stable learning theory: proves that the estimation error of a misspecified linear model is upper-bounded by O(δ/λ) — collinearity (small λ) amplifies misspecification error, and reweighting to decorrelate features (increasing λ) tightens the bound.
  • Bridging perspectives: shows that the two derivations of stable learning (causal inference via confounder balancing; statistical learning via collinearity reduction) converge on the same objective — making input variables independent — providing theoretical common ground.
  • Implications for explainability and fairness: demonstrates that stable models using direct causal variables are consistent with partial-dependence-plot interpretability and avoid fairness failures caused by spurious correlations.
  • Paradigm comparison: maps IID learning, transfer learning, domain generalization, and stable learning along axes of prior knowledge about the test distribution and diversity of training environments (Fig. 3).

① Apply stable learning's global balancing to causal-stable spatial accessibility models

Extending this paperThe paper's global balancing algorithm decorrelates input features to isolate direct causal predictors — but it is developed for IID tabular settings. A natural extension is to lift this into non-IID spatial settings where input features are geographically autocorrelated (e.g., population density, land use, income co-vary spatially). Extending the paper: introduce a spatial covariance penalty into the global balancing loss so that the reweighting also removes spatial confounding — not just distributional confounding — yielding a geographically stable predictive model.

Added value to my researchIn my 2SFCA/R2SFCA accessibility work, the same spatial confounders (density, transit access) create spurious associations between supply and health outcomes. Stable learning with spatial decorrelation would let me identify which accessibility dimensions causally predict health, making my R2SFCA coefficients more transferable across cities with different density structures — a directly actionable upgrade to spatiotemporal health geography.

② Extend stable learning's fairness implications to spatially-defined group fairness

Extending this paperThe paper links stable learning to group and individual fairness by showing that models based on direct causal features avoid spurious-correlation-induced disparities. However, it treats groups as demographically defined (race, gender). Extending the paper: define spatially-indexed groups (neighborhoods, census tracts) and embed stable learning's sample reweighting within a geographic fairness constraint — ensuring that predictions are not only stable across temporal distribution shifts but also equitable across spatial units with systematically different feature distributions (e.g., historically redlined vs. affluent neighborhoods).

Added value to my researchMy work on digital divides and telehealth access confronts exactly this structure: model performance on healthcare access prediction systematically differs across spatially defined disadvantaged groups. Stable learning with spatial-group fairness would give my explainable ML pipeline a principled causal anchor for why predictions degrade spatially — and a sample-reweighting fix that is theoretically grounded rather than post-hoc.

③ Use stable learning's covariate decorrelation as a pre-filter for MGWR/GWR bandwidth selection

Extending this paperThe paper proves that collinearity among input variables (small eigenvalue λ of the design matrix) amplifies misspecification error and is the core mechanism stable learning corrects. Geographically weighted regression (GWR/MGWR) faces an analogous problem: local bandwidth selection conflates spatial autocorrelation (geographic smoothness) with spurious correlation among covariates. Extending the paper: apply the paper's SRDO sample reweighting as a pre-decorrelation step before MGWR, so that locally fitted coefficients capture genuine spatial heterogeneity of causal effects rather than collinearity artifacts.

Added value to my researchMGWR is my core spatial regression tool. Current bandwidth selection via cross-validation optimizes fit but cannot distinguish causal spatial non-stationarity from collinearity-induced instability. A stable-learning-decorated MGWR would make my spatial regression results more reproducible across study areas and more credible for policy — directly applicable to my health geography and urban analytics work.

④ Combine stable learning with causal structure discovery to automate feature selection in spatiotemporal prediction

Extending this paperThe paper assumes that the causal structure (which variables are direct causes of Y) is at least partially known or can be approximated via balancing. Extending it with automated causal structure discovery (e.g., score-based DAG learning) would make stable learning fully data-driven: first learn which features are plausibly direct causes, then apply global balancing only over that causal skeleton — avoiding the computational cost of treating all features as potential treatments and potentially improving finite-sample stability.

Added value to my researchIn multi-source spatiotemporal fusion (mobile phone + remote sensing + social media + POIs), I face exactly the challenge of which signals causally drive health or mobility outcomes versus which are mere proxies. An automated causal pre-filter before stable learning would give my data fusion pipeline a principled variable selection step that reduces spurious signal amplification during spatial prediction — directly applicable to urban health index construction.

⑤ Embed stable learning's reweighting objective into a physics-informed graph neural network for cross-city transfer

Extending this paperThe paper proposes stable learning as a framework for out-of-distribution generalization but evaluates it primarily in vector-feature settings. Extending it: encode the paper's global balancing loss as a graph-level decorrelation constraint in a GNN operating on city street/facility graphs, so that node embeddings learned by the GNN are decorrelated from spurious spatial confounders. The paper's theoretical guarantee (Eq. 1 and the O(δ/λ) bound) would then apply at the graph level — enabling a city-graph model that is provably more stable when transferred to cities with different density or infrastructure distributions.

Added value to my researchMy physics-informed graph learning agenda aims to build transferable urban AI models. Embedding stable learning's reweighting into a GNN trained on multiple cities would give the transfer mechanism a causal rather than purely statistical justification — meaning models trained on data-rich cities would generalize to data-scarce ones with theoretically bounded degradation, directly supporting my goal of scalable GeoAI for public health policy.
