Self-supervised learning of hologram reconstruction using physics consistency

Luzhe Huang · Hanlong Chen · Tairan Liu · Aydogan Ozcan Published 2023-08-07 DOI 10.1038/s42256-023-00704-7 Source nature.com ↗
Core Conclusion

GedankenNet is a self-supervised deep neural network for hologram reconstruction that requires no experimental training data and no prior knowledge of sample types: it trains exclusively on a physics-consistency loss applied to artificially generated random images. After self-supervised training, GedankenNet successfully reconstructs experimental holograms of unseen biological specimens (human tissue sections, Pap smears) with superior external generalization over supervised models, and its outputs are physically consistent with the wave equation in free space — encoding wave propagation implicitly without ever seeing a real sample.

Why It Matters

  • Eliminates the labelled-data bottleneck in microscopy. Acquiring, aligning, and annotating experimental holographic image pairs is laborious and expensive. GedankenNet bypasses this entirely by training on synthetic random images, democratizing deep-learning microscopy for labs without large imaging datasets.
  • Superior zero-shot generalization. Unlike supervised models that overfit to training sample types (e.g., lung tissue), GedankenNet generalizes across diverse, never-seen tissue types (salivary gland, prostate, kidney, Pap smears) without any fine-tuning or transfer learning.
  • Physics law embedded in inference. The physics-consistency loss encodes the wave equation into the network, so GedankenNet outputs are compatible with Maxwell's equations — it does not hallucinate non-physical fields and remains robust to unknown perturbations (axial distance shifts, pixel-size changes, wavelength variation).
  • ~128× speed-up over classical iterative methods. GedankenNet achieves single-forward-pass inference, reconstructing 1 mm² holographic images far faster than the multi-height phase retrieval (MHPR) algorithm it outperforms in quality.
  • Broadly applicable inverse-problem framework. The physics-consistency self-supervision principle is not specific to holography — it opens a path to label-free training for any imaging modality where a differentiable physical forward model exists.

Core Methods

  • Physics-consistency self-supervised loss: instead of comparing network output to ground-truth images, GedankenNet propagates the output complex field forward using the angular spectrum approach and compares the predicted hologram to the input hologram. The loss combines Fourier Domain Mean Absolute Error (FDMAE) and MSE with a total variation (TV) regularization term.
  • GedankenNet architecture: a series of Spatial Fourier (SPAF) blocks with a global residual connection and two 1×1 convolution layers. Each SPAF block contains recursive SPAF modules that perform both spatial convolution and a learnable linear transform in the Fourier domain (pixel-wise multiplication with trainable weight tensors), using a pyramid-like window scheme to handle multi-scale spatial frequencies.
  • Angular spectrum propagation (ASP) forward model: the free-space propagation is implemented as a differentiable layer using FFT, enabling gradient back-propagation through the physical forward model for self-supervised learning.
  • Artificial training data synthesis: random complex fields (from randimage) are propagated by ASP at M designated axial distances to generate synthetic multi-height holograms. No real-world structure or sample-specific features are needed.
  • Evaluation metrics: SSIM and RMSE (on amplitude and phase separately); Enhanced Correlation Coefficient (ECC) on complex fields — all computed against MHPR ground-truth fields retrieved from M = 8 holograms.
  • Baselines compared: supervised Fourier Imager Network (FIN), supervised U-Net, iterative MHPR, and deep image prior (DIP) — all benchmarked on identical test sets.
  • GedankenNet-Phase / GedankenNet-Phase-λ variants: phase-only output extensions using dynamic SPAF (dSPAF) modules with dense links, enabling autofocusing at unknown axial distances and robustness to unknown illumination wavelengths.
