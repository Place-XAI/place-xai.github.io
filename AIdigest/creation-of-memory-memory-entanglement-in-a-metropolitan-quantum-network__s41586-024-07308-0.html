Creation of memory–memory entanglement in a metropolitan quantum network

Jian-Long Liu · Xi-Yu Luo · Yong Yu · Chao-Yang Wang · Bin Wang · Yi Hu · Jun Li · Ming-Yang Zheng · Bo Yao · Zi Yan · Da Teng · Jin-Wei Jiang · Xiao-Bing Liu · Xiu-Ping Xie · Jun Zhang · Qing-He Mao · Xiao Jiang · Qiang Zhang · Xiao-Hui Bao · Jian-Wei Pan Published 2024-05-15 DOI 10.1038/s41586-024-07308-0 Source nature.com ↗
Core Conclusion

This work demonstrates the first memory–memory entanglement across a real metropolitan area, using three independent atomic-ensemble quantum memory nodes separated by up to 12.5 kilometres of deployed fiber. A central photonic server heralds entanglement via single-photon detection, and active phase stabilization keeps the network coherent. Concurrent pairwise entanglement is achieved and the memory lifetime exceeds the round-trip communication time — establishing a city-scale testbed for multi-node quantum network protocols and marking the start of practical quantum internet research.

Core Methods

  • Atomic ensemble quantum memory (DLCZ protocol): each node stores a single collective spin excitation in a cold rubidium atomic ensemble via the Duan–Lukin–Cirac–Zoller (DLCZ) scheme, enabling probabilistic but heralded photon–memory entanglement generation.
  • Telecom wavelength conversion: the memory photon at 795 nm is frequency-upconverted/downconverted to the telecom C-band (~1342 nm) via difference-frequency generation in periodically poled lithium niobate (PPLN) waveguides, enabling low-loss transmission over standard installed fiber.
  • Active weak-field phase stabilization: a dedicated phase-locking protocol continuously tracks and corrects fiber-induced phase fluctuations using a reference laser at the photonic server, maintaining the entanglement phase across the metropolitan fiber links.
  • Heralded entanglement generation: coincidence detection of a single telecom photon at the photonic server heralds the creation of a remote memory–memory entangled state (Bell state) between a pair of nodes. Success probability and fidelity are measured via quantum state tomography.
  • Quantum state tomography: the entangled state of the two remote memory qubits is reconstructed by measuring correlations in multiple bases, yielding state fidelity F and entanglement witness values.
  • Concurrent entanglement: the network is operated so that entanglement between any two node pairs can be generated simultaneously, demonstrating multi-pair operation essential for quantum repeater chains.
