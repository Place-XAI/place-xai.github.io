Iterative variational learning of committor-consistent transition pathways using artificial neural networks

Alberto Megías · Sergio Contreras Arredondo · Cheng Giuseppe Chen · Chenyu Tang · Benoît Roux · Christophe Chipot Published 2025-06-30 DOI 10.1038/s43588-025-00828-3 Source nature.com ↗
Core Conclusion

The PCCANN (path-committor-consistent artificial neural network) simultaneously learns the committor function and a committor-consistent string (CCS) — a reaction pathway that is by construction aligned with the true ensemble of reactive transition paths. Operating via a finite time-lag variational principle, it surpasses methods limited to the overdamped diffusive limit, correctly identifies multiple competing pathways through clustering, and robustly recovers established rate constants and transition mechanisms on benchmark potentials (BS, Müller–Brown, triple-well), a peptide (NANMA isomerization), and a mini-protein (chignolin folding). Its architecture-resilient design and adaptability to collective variables make it a general-purpose tool for enhanced-sampling simulations of rare events in biomolecular systems.

Why It Matters

  • Resolves the chicken-and-egg problem in path-finding. The correct reaction pathway requires knowing the committor, but learning the committor needs good sampling along the pathway. PCCANN solves both simultaneously within a single iterative framework.
  • Extends beyond the diffusive limit. Traditional variational committor approaches require infinitesimal time lags (valid only for overdamped Brownian dynamics). The finite time-lag variational principle used here works across underdamped Langevin dynamics and more complex regimes, broadening applicability considerably.
  • Handles multiple competing pathways. By combining committor learning with K-means/K-medoids clustering, the method identifies and separately characterizes equiprobable competing reaction tubes — a capability absent from most prior ANN-based committor learners.
  • Reproduces ground-truth kinetics. Rate constants inferred from the converged committor agree with mean-first-passage-time benchmarks and SMwST references on all tested systems, validating the physical fidelity of the learned pathway.
  • Architecture-resilient and open-source. The same four-dense-layer, 32-neuron ANN serves every system tested regardless of dimensionality; all code and trajectories are publicly deposited on Code Ocean.

Core Methods

  • Variational committor learning (VCN): minimize the two-point committor time-correlation function Cqq(τ) = ½⟨(q(τ)−q(0))²⟩ with boundary constraints, using a Siamese ANN. Unlike prior approaches, the finite time lag τ avoids the infinitesimal-lag / overdamped-diffusion assumption.
  • Committor-consistent string (CCS) learning: represent the string as a Gaussian-softmax expansion qs(z) whose coefficients b are solved by least squares from q values at image positions, enforcing alignment of the string tangent with ∇qs through a dedicated alignment loss Ltr. A separate equidistance loss Leq maintains uniform image spacing.
  • Iterative PCCANN protocol: (1) run biased MD along an initial path guess; (2) learn committor and CCS; (3) compute the committor-averaged path (CAP) to form an improved initial guess; (4) iterate to convergence. Biased-trajectory contributions are unbiased via a reweighting scheme based on the bias potential δW.
  • Multiple-pathway detection: cluster trajectories in committor-stratified isocommittor slices using K-means (committor-based) or K-medoids (chignolin); compute a separate CAP and launch an independent iterative procedure for each identified reaction tube.
  • Enhanced sampling: WTM-eABF (well-tempered meta-extended adaptive biasing force) is used as the biasing engine along the path-collective variable (PCV) defined by the CCS at each iteration, implemented via NAMD + Colvars.
  • Architecture: Siamese feed-forward ANN with 4 hidden layers × 32 neurons (ELU activation), sharing weights for the two time-lag inputs. An external optimizer layer handles the string image positions. Total loss = Lq + Ls with hyperparameters kb, keq, ktr.

① Replace hand-picked collective variables with a CV-free committor learned directly from atomic coordinates

Extending this paperPCCANN's central limitation, acknowledged by the authors, is its dependence on user-chosen collective variables (CVs) — wrong CV choices yield a misleading committor and CCS. A corpus paper published just months later, "Learning the committor without collective variables" (Nature Computational Science, 2026), demonstrates a geometric vector perceptron graph neural network that predicts the committor directly from all-atom coordinates with no a-priori CV selection. Integrating this CV-free committor head into the PCCANN iterative loop would allow the string images to be placed in full Cartesian space, making the pipeline genuinely CV-agnostic and removing the expert knowledge bottleneck that currently limits applicability to well-studied proteins.

② Encode the molecular energy landscape as a learnable interatomic potential to close the simulation-learning loop

Extending this paperThe PCCANN currently relies on pre-specified force fields (NAMD + CHARMM) to generate trajectories; the learned committor feeds back only through the PCV biasing term. A logical extension is to jointly train a machine-learned interatomic potential (MLIP) — such as the universal M3GNet-type graph deep learning IAP reported in the corpus — alongside the PCCANN. This would allow the CCS to adaptively guide sampling on a continuously refined energy surface, progressively correcting force-field errors while learning the true transition pathway. The result would be a self-consistent enhanced-sampling / potential-refinement loop that is especially valuable for novel biomolecules where reference force fields are unreliable.

④ Transfer the iterative committor-string idea to metastable-state detection in urban mobility networks

Extending this paperPCCANN's core abstraction — two metastable basins A and B, a reaction coordinate characterizing the probability of transitioning from A to B, and an iterative path-finding procedure that aligns the string with the committor gradient — is mathematically domain-agnostic. The paper demonstrates this on analytic potentials and biomolecules, but the same formalism could be applied to non-equilibrium stochastic dynamics on a network. Extending PCCANN to a graph-structured state space (e.g., a city's POI graph) would allow identification of the most probable "transition pathways" between urban attractors (home, work, healthcare, commercial) — reinterpreting the committor as the probability that a mobility trajectory starting at a given location reaches a target zone before any other. The iterative clustering step maps cleanly onto the detection of competing mobility corridors.
