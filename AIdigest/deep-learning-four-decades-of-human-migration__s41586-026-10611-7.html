Deep learning four decades of human migration

Thomas Gaskin · Guy J. Abel Published 2026-06-10 DOI 10.1038/s41586-026-10611-7 Source nature.com ↗
Core Conclusion

An ensemble of deep recurrent neural networks produces the first annual, globally consistent origin-destination migration dataset spanning 230 countries and regions from 1990 to 2023, outperforming all existing five-year stock-differencing methods on held-out data. By integrating UN migrant stocks, QuantMig harmonized flows, national statistics, and Facebook digital-trace estimates, the framework captures both long-term trends and short-term shocks — including COVID-19 and armed conflicts — while propagating uncertainty to generate calibrated confidence bounds that expose data-poor regions.

Existing resources provide estimates only at five-year intervals for a subset of corridors; this work extends temporal resolution to individual years for all 230 UN-recognized countries, revealing dynamics previously smoothed away.
  • Paradigm shift from indirect stock differencing to learned flows. The paper replaces decades of demographic accounting heuristics with a recurrent neural network that maintains memory of past migration regimes — a fundamental methodological advance for migration science.

Core Methods

  • Deep recurrent neural network ensemble: 15 independently initialized networks (7 layers, 60 neurons/layer, tanh activation + CeLU output), each maintaining a 100-dimensional latent state across time to capture migration memory and temporal correlation.
  • Demographic accounting framework: flows Tbij are constrained to satisfy the migrant-stock evolution ODE (births, deaths, migration balance), ensuring physical consistency between estimated flows and observed stocks.
  • Multi-source loss function: L2 loss jointly over stock changes (ΔS), net migration (M), and bilateral flows (F), each term weighted by inverse uncertainty; QuantMig's published standard errors directly populate flow-loss weights.
  • Symmetrized Yeo-Johnson transform: all skewed input and target distributions are pre-transformed to approximate normality (λ = 0.7 for targets) to prevent large-flow dominance in gradient updates.
  • Uncertainty quantification via ensemble + pushforward: 1,500 samples generated by combining 15 network parameter distributions × 100 initial-stock uncertainty draws; uncertainty propagated from inputs through the trained network.
  • Fivefold cross-validation on corridors: flow corridors split into 5 equal folds; five separate networks trained on 4-fold subsets; achieves 94% Pearson R on training flows and 73% on held-out test flows.
  • Elasticity analysis: sensitivity of model predictions to each covariate computed analytically via the chain rule through the Yeo-Johnson transform; life expectancy and GDP per capita identified as most influential covariates.
  • Optimal initial-stock calibration: least-squares computation of a constant offset ρbi that aligns modelled stock trajectories with observed data while accounting for cumulative mortality.

Extending this paperThe paper uses a fixed, pre-specified covariate vector (GDP, life expectancy, religious similarity, distance, etc.) — essentially a hand-crafted feature set fed into the RNN. Importantly, the paper's uncertainty ensemble would then propagate learned-embedding uncertainty rather than just covariate noise.
