Multimodal language and graph learning of adsorption configuration in catalysis

Janghoon Ock · Srivathsan Badrinarayanan · Rishikesh Magar · Akshay Antony · Amir Barati Farimani Published 2024-11-27 DOI 10.1038/s42256-024-00930-7 Source nature.com ↗
Core Conclusion

A multimodal pretraining strategy ("graph-assisted pretraining") aligns text embeddings from CatBERTa with graph embeddings from EquiformerV2 via self-supervised contrastive loss, reducing MAE in adsorption energy prediction by 7.4–9.8%. A fine-tuned generative LLM (CrystaLLM) can then produce indicative crystal structure files from only chemical composition and surface orientation, enabling energy predictions for catalyst systems without exact atomic coordinates and more than doubling the likelihood of predictions falling within the DFT energy range.

Why It Matters

  • Bridges accuracy and accessibility in chemistry ML. GNNs are precise but demand exact atomic positions unavailable from experiments; language models accept text but sacrifice accuracy. This work forges a principled bridge, delivering GNN-calibrated precision while retaining the input flexibility of language models.
  • Opens experiment-driven catalyst screening. Because chemical composition and surface orientation are experimentally accessible without full structural characterization, the proposed framework can screen catalysts from experimental data — a capability GNN-only pipelines cannot offer.
  • Self-supervised and label-free pretraining. Graph-assisted pretraining requires no DFT energy labels during the alignment phase, meaning pretrained checkpoints can be shared and reused across downstream tasks without additional costly quantum-chemical calculations.
  • Generative + predictive in one pipeline. Combining CrystaLLM (structure generation) with CatBERTa (energy prediction) demonstrates a two-stage LLM pipeline for materials property prediction, pointing toward autonomous AI-driven catalyst design workflows.

Core Methods

  • Graph-assisted pretraining (self-supervised multimodal alignment): symmetric cross-entropy contrastive loss (CLIP-style) maximizes cosine similarity between matched text–graph embedding pairs; EquiformerV2 graph encoder is frozen; CatBERTa projects its <s> token via a linear head to match 3,200-dim graph embeddings. No DFT labels required at pretraining stage.
  • Text encoder — CatBERTa: RoBERTa-base fine-tuned for catalyst energy regression. Input string has three sections: (1) adsorbate chemical symbol; (2) catalyst composition + Miller index; (3) adsorption configuration (primary and secondary interacting atoms identified by Pymatgen with covalent-radius cutoffs).
  • Graph encoder — EquiformerV2: SE(3)/E(3)-equivariant GNN with eSCN convolutions, attention renormalization, separable S2 activation, and separable layer normalization. Max-pooled across atom node embeddings to a single 3,200-dim system embedding (state-of-the-art: MAE 0.22 eV S2EF, 0.31 eV IS2RE on OC20).
  • Generative structure prediction — CrystaLLM: GPT-2-based autoregressive LLM (16 layers, 16 heads, block size 2,048) pretrained on inorganic crystal CIFs; fine-tuned on OC20 + OC20-Dense CIFs for 6,000 epochs (lr = 6×10−4). Generates CIFs conditioned on adsorbate + catalyst composition + Miller index alone.
  • Structure-to-text conversion: Pymatgen-based neighbor detection (covalent-radius cutoff) identifies primary and secondary interacting atoms; for LLM-generated CIFs a relaxed 4× cutoff multiplier accommodates indicative (non-exact) structures.
  • Evaluation metric — Prediction Inclusion Ratio (PIR): fraction of model predictions falling within the DFT energy range of all adsorption configurations for a given pair, extended by the model's intrinsic error threshold (standard deviation of predictions on actual configuration strings).

⊚ Generalize graph-assisted pretraining as a domain-agnostic multimodal alignment recipe

Extending this paperThe paper demonstrates graph-assisted pretraining exclusively in the atomistic catalysis domain. A direct extension: replace CatBERTa + EquiformerV2 with any text encoder + graph encoder pair and test whether the CLIP-style contrastive alignment holds across molecular, polymer, and drug-like chemical spaces. The finding that pretraining on OC20 benefits OC20-Dense fine-tuning despite no dataset overlap suggests genuine cross-domain transferability — but this is reported as an observation rather than systematically validated. A follow-on paper could benchmark the same pretraining recipe across multiple property prediction tasks to characterize when and why cross-domain transfer succeeds.

Added value to my researchThe multimodal alignment principle translates directly to my physics-informed graph learning agenda: align a geospatial text encoder (describing neighborhoods, POI types, transport links as text) with a spatial GNN (encoding street/facility graph topology) in a self-supervised fashion. Contrastive pretraining on unlabeled geographic text–graph pairs could prime the GNN for downstream accessibility tasks with scarce labeled data — directly applicable to telehealth digital-divide prediction and primary-care accessibility delineation where labeled outcome data is limited.

⊚ Replace the frozen GNN teacher with a foundation interatomic potential for richer transfer

Extending this paperThe paper freezes EquiformerV2 as the graph teacher. Recent work demonstrates that "Platonic" foundation machine learning interatomic potentials (MLIPs) trained on diverse datasets exhibit convergent internal representations across architectures. Extending this paper: substitute the frozen task-specific GNN teacher with a foundation MLIP and examine whether the broader representation further reduces MAE on out-of-domain catalyst systems — and whether the text encoder inherits the foundation model's cross-material transferability. This would also test whether the graph-assisted pretraining framework can be decoupled entirely from a single GNN checkpoint.

Added value to my researchThis mirrors my interest in fine-tuning geospatial foundation models: a geospatial foundation model (trained on global remote sensing + street graphs) provides richer spatial priors than a locally trained GNN, just as a foundation MLIP provides richer structural priors than a domain-specific GNN. The teacher–student contrastive alignment pattern from this paper offers a concrete recipe for efficiently adapting a geospatial foundation model to local urban health tasks using limited labeled data — a key bottleneck in health geography research.

⊚ Extend PIR to a calibrated conformal uncertainty quantification framework

Extending this paperThe paper introduces the Prediction Inclusion Ratio (PIR) as a task-specific metric for whether LLM-derived strings help predictions fall within the DFT energy range. However, PIR treats the intrinsic error threshold as fixed and does not propagate the LLM generation uncertainty (CIF validity rate, composition match tolerance) through to the energy prediction. Extending the paper: couple ensemble prediction or split conformal prediction to quantify how CIF generation uncertainty propagates to confidence intervals on the final energy prediction — producing calibrated coverage guarantees rather than a binary in/out-of-range verdict.

Added value to my researchCalibrated uncertainty quantification is central to my explainable ML for spatial prediction work. The two-stage pipeline (generative input → predictive output) is structurally identical to imputation-then-prediction workflows in spatial accessibility modeling (e.g., imputing missing facility locations, then predicting service coverage under uncertainty). Applying conformal prediction within this pipeline would allow my models to report spatially explicit coverage guarantees — a methodological contribution suited to policy-facing health geography, where decision-makers require reliable confidence bounds.

⊚ Use the three-section text schema as a structured prompt template for geospatial LLMs

Extending this paperA subtle but powerful contribution is the design of the three-part input string (adsorbate | catalyst + Miller index | adsorption configuration) that decomposes the physical system into interpretable textual sections. The attention analysis (Fig. 3) shows that this structured decomposition interacts measurably with model attention — graph-assisted pretraining redirects focus specifically to the configuration section. Extending the paper: apply this section-aware pretraining to richer chemical text descriptions (SMILES, property keywords, synthesis conditions) and systematically study which decomposition granularity optimally captures domain priors.

Added value to my researchThe structured three-section schema translates into structured prompts for geospatial LLMs: decompose a location description into (1) the target entity (patient, household); (2) the spatial context (neighborhood density, road access type); (3) the service configuration (nearest facility type, travel time, capacity ratio). Attention analysis on this decomposition would reveal which sections geospatial models actually use for accessibility prediction — providing a new explainability probe analogous to Fig. 3, directly applicable to my 2SFCA/R2SFCA research for validating model attention toward the service-area interaction term.

⊚ Scale the generative–predictive pipeline to multi-objective catalyst design via Pareto-aware sampling

Extending this paperThe paper predicts a single property (adsorption energy) from generated structures. Extending it: use the generative CrystaLLM stage to sample diverse candidate CIF configurations, then predict multiple catalyst properties simultaneously (adsorption energy, activation barrier, selectivity) via multi-task fine-tuning of CatBERTa — and guide CrystaLLM generation toward the Pareto front of these properties using preference-conditioned prompting. This transforms the framework from a single-property surrogate into a multi-objective catalyst design engine, directly aligned with the paper's stated goal of "a comprehensive, language-based platform for catalyst design."
