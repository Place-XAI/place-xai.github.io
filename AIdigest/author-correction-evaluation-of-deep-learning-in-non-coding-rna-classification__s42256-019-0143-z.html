Core Conclusion

This notice records an author correction to the original research article "Evaluation of deep learning in non-coding RNA classification" (10.1038/s42256-019-0051-2, Nature Machine Intelligence, 2019). The corrected article demonstrated that deep learning architectures — particularly CNNs — significantly outperform classical machine learning methods for ncRNA family classification across multiple benchmark datasets, while also exposing that imbalanced and homology-redundant training sets can inflate reported performance metrics, calling for stricter evaluation protocols in computational genomics.

Core Methods

  • Convolutional neural networks (CNNs) applied directly to one-hot or k-mer encoded nucleotide sequences, learning motif-level features for ncRNA family classification without manual feature engineering.
  • Recurrent neural networks (RNNs / LSTMs) for capturing long-range sequential dependencies in RNA sequences.
  • Classical baselines: support vector machines, random forests, and gradient boosting applied to hand-crafted sequence features for head-to-head comparison against deep learning.
  • Benchmark design: systematic train/test splitting with attention to sequence redundancy removal (to prevent homology leakage) and class imbalance correction, establishing a methodological checklist for ncRNA evaluation.
  • Performance metrics: accuracy, F1, AUC — with analysis of how class imbalance affects each metric differentially, guiding metric selection for future ncRNA studies.

Extending this paperThe corrected article shows that CNNs outperform classical methods but provides no interpretability of which learned sequence patterns drive classification.
