Real-time inference for binary neutron star mergers using machine learning

Maximilian Dax · Stephen R. Green · Jonathan Gair · Nihar Gupte · Michael Pürrer · Vivien Raymond · Jonas Wildberger · Jakob H. Macke · Alessandra Buonanno · Bernhard Schölkopf Published 2025-03-05 DOI 10.1038/s41586-025-08593-z Source nature.com ↗
Core Conclusion

DINGO-BNS is a machine-learning framework based on simulation-based inference with normalizing flows that performs complete Bayesian parameter estimation of all 17 binary neutron star parameters in just 1 second — three orders of magnitude faster than conventional methods — without sacrificing accuracy. It enables pre-merger sky localization with ~30% improved precision over existing low-latency approximations, and scales to signals up to an hour in length, making it a blueprint for next-generation gravitational-wave detectors.

Core Methods

  • Simulation-based inference (SBI) / Neural posterior estimation (NPE): a conditional normalizing flow, q(θ|d), is trained via maximum likelihood on simulated (θ, d) pairs to approximate the Bayesian posterior p(θ|d). Once trained, sampling is instantaneous for any new observation.
  • Prior conditioning: a novel technique that trains a single neural network conditioned on a hyperparameter ρ (the reference chirp mass), enabling the model to be tuned at inference time to event-specific, narrow priors — effectively amortizing training cost across the entire prior volume without retraining.
  • Heterodyning + frequency multibanding (data compression): the predominant phase evolution is factored out (heterodyning), reducing signal oscillations by orders of magnitude; then frequency multibanding coarsens resolution adaptively across bands, achieving compression factors of 60–650× without loss of information — reducing input dimensionality from ~107 to manageable size.
  • Frequency masking for pre-merger inference: the frequency-domain data stream is masked above a cutoff frequency corresponding to a chosen pre-merger time, enabling inference at any moment before coalescence.
  • Importance sampling correction (DINGO-IS): samples from the approximate neural posterior are reweighted using the exact GW likelihood, providing asymptotically exact Bayesian results and a diagnostic sample efficiency metric (typically 31–79% in experiments).
  • JAX waveform/likelihood implementation: just-in-time compiled JAX code enables ~0.19 s for 50,000 likelihood evaluations on an H100 GPU, completing the full inference pipeline in under 1 s.
  • EOS likelihood via parameter conditioning: DINGO-BNS can be conditioned on fixed sky position or mass/tidal parameter subsets, enabling fast equation-of-state comparison via Monte Carlo integration of the conditional evidence p(d|EOS) in 1–3 s.

① Transfer prior conditioning to spatiotemporal density estimation

Extending this paperThe paper's key innovation — prior conditioning — allows a single trained network to operate under different, event-specific prior distributions selected at inference time. This avoids retraining while retaining accuracy across a wide prior volume. A direct extension is to apply this principle to other scientific domains where the prior over key parameters is known to be tightly constrained for individual instances (e.g., known chirp-mass analogs in other fields). The technique could be formally generalized as a hyperprior amortization framework: train once on a hyperprior, condition on instance-level auxiliary information at inference time, enabling zero-shot adaptation without sequential retraining.

Added value to my researchIn my spatiotemporal kernel density estimation (KDE) work, bandwidth selection is typically fixed or cross-validated globally. Prior conditioning maps directly onto adaptive bandwidth selection conditioned on local data density: a normalizing-flow KDE trained with a hyperprior over bandwidths could be tuned at inference time to the local density regime of each spatial unit — analogous to DINGO-BNS tuning to each event's chirp mass. This would yield a GeoAI-native, amortized R2SFCA where the service-area radius adapts to each facility's local context without re-solving an optimization problem.

② Adopt heterodyning + multibanding as a general physics-informed data compression pipeline

Extending this paperDINGO-BNS achieves 60–650× lossless compression by factoring out known physical structure (the chirp phase) before applying adaptive spectral decimation. This is a general recipe: identify the dominant analytic behavior of a signal, factor it out, then apply data-adaptive compression to the residual. The paper could be extended by formalizing this as a transferable physics-informed compression layer that learns what structure to factor out end-to-end — rather than hand-coding the post-Newtonian chirp formula — enabling the same approach in domains where no closed-form dominant behavior is known a priori.

④ Apply importance-sampling validation as a universal accuracy guarantee for ML inference pipelines

Extending this paperDINGO-BNS's key reliability mechanism is coupling the neural posterior approximation q(θ|d) with importance sampling reweighting using the exact likelihood — producing a sample-efficiency diagnostic that measures how well the neural network has learned the posterior. Papers could extend this by studying how sample efficiency degrades as a function of (a) out-of-distribution data, (b) model capacity, and (c) training data size — turning what is currently an ad-hoc check into a principled calibration theory for simulation-based inference, including formal bounds on the KL divergence from the true posterior.

Extending this paperDINGO-BNS is designed to run continuously on the incoming data stream, scanning for events by sliding the coalescence-time prior over the data in real time, triggering alerts when the inferred SNR or Bayes factor exceeds a threshold.
