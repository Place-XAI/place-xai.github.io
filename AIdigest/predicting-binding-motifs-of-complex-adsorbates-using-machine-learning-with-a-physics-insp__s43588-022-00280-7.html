Predicting binding motifs of complex adsorbates using machine learning with a physics-inspired graph representation

Wenbin Xu · Karsten Reuter · Mie Andersen Published 2022-07-25 DOI 10.1038/s43588-022-00280-7 Source nature.com ↗
Core Conclusion

A data-efficient, physics-inspired machine learning model — the Wasserstein Weisfeiler–Lehman graph kernel combined with Gaussian process regression (WWL-GPR) — predicts adsorption enthalpies and binding motifs of complex adsorbates (including bi- and higher-dentate modes) at transition metal catalysts with an r.m.s.e. of ~0.18–0.2 eV, far outperforming vector-based models. Trained on only ~1,700 DFT data points, the model generalizes to bimetallic alloys and out-of-domain metals, and an ensemble uncertainty approach reliably flags high-error predictions, making it well-suited for active-learning catalyst screening.

Why It Matters

  • Enables catalyst screening beyond simple adsorbates. Most ML models for catalysis handle only monodentate species; this is one of the first data-efficient frameworks that simultaneously predicts which binding motif a complex molecule (up to 9 atoms, with mono/bi/higher-dentate modes) adopts and how strongly it binds.
  • 300× smaller training set than the Open Catalyst Project. By embedding physically motivated features (d-band moments, HOMO/LUMO levels, SOAP descriptors) as node attributes, the model achieves practical accuracy (~0.18 eV r.m.s.e.) with ~1,700 data points versus ~500,000 in purely geometry-driven neural approaches.
  • Extrapolates to alloys and new elements. WWL-GPR achieves r.m.s.e. of 0.23 eV on an unseen CuCo alloy and an out-of-domain element (Ru) — a necessary capability for computational catalyst discovery at scale.
  • Provides principled uncertainty quantification. An ensemble of 100 bootstrapped GPR models reliably assigns high uncertainty to high-error predictions, enabling reliable active-learning workflows where costly DFT calculations are directed where the model is least confident.
  • Fully open and reproducible. All data, code, and DFT coordinates are released on GitHub and Zenodo, enabling the community to build on or benchmark against the model.

Core Methods

  • WWL graph kernel: the Wasserstein Weisfeiler–Lehman kernel computes similarity between surface–adsorbate graphs by iteratively updating node embeddings with neighbor information, then computing the Wasserstein (earth-mover) distance between node-attribute distributions — allowing continuous-valued, physically motivated attributes unlike classical WL kernels.
  • Physics-inspired node attributes: surface atoms carry d-band moments (center, width, skewness), density of states at the Fermi level, and work function from clean-surface DFT; adsorbate atoms carry HOMO/LUMO energy levels and SOAP descriptors capturing local geometry — all precomputed without the adsorbate present.
  • Surface-adsorption hyperparameters: three edge-weight classes (adsorbate–adsorbate, surface–surface, adsorbate–surface) plus inner/outer radial cutoffs control the emphasis placed on atomic shells around the active site — functioning as a physically motivated attention mechanism.
  • Gaussian process regression (GPR): the WWL kernel feeds a GPR model with a probabilistic posterior, providing both a point prediction and a standard deviation. Hyperparameters tuned separately for interpolation (tight) vs. extrapolation (loose regularization) regimes.
  • Comparative baselines: SISSO (symbolic regression, 8D rung-3 descriptor), RBF-GPR (vector input, averaged surface/adsorbate features), and XGBoost (gradient-boosted trees) evaluated with fivefold stratified cross-validation on both databases.
  • Ensemble uncertainty quantification: 100 bootstrapped GPR models produce an ensemble standard deviation that better identifies high-error predictions than the single-model posterior — validated through calibration, sharpness, and dispersion metrics.
  • Kernel PCA: non-linear dimensionality reduction of the learned kernel matrix reveals clustering by metal type and adsorbate identity, confirming the model captures chemically meaningful latent structure.

① Replace the WWL graph kernel with a learnable heterogeneous GNN that inherits its physics priors

Extending this paperWWL-GPR achieves data efficiency through hand-crafted physics features (d-band moments, HOMO/LUMO, SOAP) embedded as fixed node attributes and manually tuned edge-weight hyperparameters that distinguish three bond classes. The natural extension is to replace the fixed WWL kernel with a physics-infused heterogeneous graph neural network that treats adsorbate–adsorbate, surface–surface, and adsorbate–surface edges as distinct typed channels — the same structural hierarchy the WWL edge-weight hyperparameters currently encode manually — while retaining the physics node features. This would let the model learn which node-pair interactions matter from data rather than hand-tuning cutoff radii. The paper's KPCA (Fig. 4) shows the kernel already captures chemically structured latent space, suggesting a GNN with the same inductive biases could generalize better to broader material classes (e.g., metal oxides) without feature re-engineering.

Added value to my researchThis is a direct analog of my physics-informed graph learning agenda in GeoAI: just as the paper bakes d-band physics into node attributes for catalysis, I embed spatial physics priors (gravity decay, boundary constraints) into GNN node/edge features for geospatial prediction. The heterogeneous-edge formulation — where edge type encodes the physical relationship between nodes — maps onto multi-modal urban graphs where road-type, transit-mode, or social-tie edges each carry distinct physics. Learning to weight edge classes rather than fixing them could improve my spatiotemporal kernel density and R2SFCA distance-decay models in a principled, data-adaptive way.

② Build a closed active-learning loop coupling ensemble uncertainty with on-demand DFT for alloy catalyst discovery

Extending this paperThe paper proposes ensemble uncertainty quantification (Fig. 5) and explicitly envisions it for active learning, but demonstrates only a single-shot evaluation without a closed feedback loop. The direct extension is to connect WWL-GPR's ensemble standard deviation to an acquisition function (expected improvement or uncertainty sampling) that iteratively selects the next DFT calculation from the combinatorially large alloy space. Crucially, the paper already shows that interpolation and extrapolation hyperparameters differ: a closed loop could dynamically switch hyperparameter regimes depending on whether the next candidate falls inside or outside the sampled chemical space, combining the paper's exploit (tight interpolation) and explore (loose regularization) settings into a unified adaptive protocol.

Added value to my researchThis active-learning-with-uncertainty template transfers directly to my data-efficient GeoAI agenda: treating expensive field surveys, POI audits, or healthcare-access ground-truth labels as the "DFT calculations" and spatiotemporal ML models as the surrogate, an ensemble uncertainty signal can direct sparse data collection to urban areas where model predictions are least reliable — exactly the active spatial sampling problem underlying my 2SFCA accessibility work in data-scarce regions and the FAIR data collection challenge in mobile phone and social media research.

③ Extend multi-dentate binding-motif prediction to transition-state and reaction-path energetics

Extending this paperWWL-GPR predicts the energetics of discrete adsorbed-state minima, but the Discussion explicitly acknowledges it "does not predict the entire potential energy surface." The most impactful extension of the paper is to lift it from stable minima to transition-state saddle points: represent the transition state as a graph where breaking/forming bonds appear as fractional edge-weight attributes encoding bond order, feed it through the WWL kernel, and train on a database of DFT-NEB (nudged elastic band) barriers. The paper's extrapolation protocol — retraining with minimal atomic-species energies for a new metal — could serve as a transfer-learning seed for barrier prediction on unseen alloys, making the framework useful for full microkinetic modeling of complex ethanol synthesis pathways.

④ Replace hand-crafted features with multimodal language + graph representations for zero-shot catalyst generalization

Extending this paperWWL-GPR's node attributes require expert-designed DFT-derived features (d-band moments, HOMO/LUMO, SOAP) that must be recomputed for every new material class, limiting generalization beyond transition metals to oxides, nitrides, or carbides. A direct extension is to augment or replace these hand-crafted attributes with pretrained chemical language model embeddings (e.g., from a materials-domain LLM) combined with learned graph embeddings. This multimodal representation — text describes what the atom is chemically, graph describes how it connects structurally — could allow the model to generalize to material families described in the literature but absent from the DFT training set, without recomputing PDOS or SOAP. The paper's own comparison (SISSO vs. GPR) shows interpretable features have value; language embeddings could provide interpretable generalizations without manual feature engineering.

Added value to my researchThis multimodal fusion idea resonates with my work on multi-source data fusion (mobile phone, social media, remote sensing, POIs): just as combining text-based semantic signals with graph-topological signals improves catalyst property prediction, combining semantic descriptions of urban amenities (text reviews, service categories) with their spatial-graph embedding could improve healthcare-accessibility models where POI completeness is uneven across regions. The key methodological insight — language embeddings regularize graph learning in data-sparse regimes — applies across both domains and could inform my fine-tuning of geospatial foundation models.

⑤ Transfer the Wasserstein graph-similarity metric to principled cross-city transferability assessment in GeoAI

Extending this paperThe WWL kernel's central innovation is computing graph similarity via the Wasserstein distance between node-attribute distributions — a metric that respects the geometry of the attribute space rather than just counting common subgraphs. The paper uses this to smoothly interpolate between similar surface–adsorbate configurations. An underexplored extension: use the same Wasserstein graph distance to build a metric space of catalysts in which the model's extrapolation performance can be predicted a priori — that is, before running any DFT — based on how far a new catalyst lies from the training set in Wasserstein distance. This would turn the paper's empirical extrapolation tests (Table in Supplementary) into a theoretically grounded transferability criterion, allowing practitioners to decide before committing DFT resources whether a given alloy is close enough to the training distribution for reliable prediction.

Added value to my researchThe Wasserstein graph distance as a transferability criterion maps directly onto my cross-city GeoAI challenge: instead of asking "how far is a new alloy from the training metals?", ask "how far is a new city's spatial graph (road network topology + POI density field) from the training cities?" This gives a principled, geometry-aware answer to whether a healthcare-accessibility model or 15-minute-city scoring trained on large US cities will transfer reliably to a mid-size city in Southeast Asia — a question currently answered only by trial-and-error fine-tuning or domain expert judgment. Coupling this distance metric with my physics-informed graph learning work would yield a theoretically grounded geospatial transfer-learning criterion for urban AI models.
