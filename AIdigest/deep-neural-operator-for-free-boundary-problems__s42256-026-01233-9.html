Deep neural operator for free boundary problems

Zongjia Long · Qi Zhou · Aiqing Zhu · Dong Dai · Yiqun Liu Published 2026-05-21 DOI 10.1038/s42256-026-01233-9 Source nature.com ↗
Core Conclusion

The Free Boundary Neural Operator (FBNO) is a universal neural operator framework that — for the first time — solves partial differential equations on a priori unknown, evolving spatial domains. By exploiting topological conjugacy between a free boundary dynamical system and a conjugate system on a fixed reference domain, FBNO learns both the domain diffeomorphism and the solution field simultaneously, achieving relative L2 errors below 1% while running up to 104 times faster than traditional numerical solvers. A formal approximation theorem proves the method is theoretically sound.

Why It Matters

  • Breaks a fundamental barrier in neural operator theory. Existing frameworks (DeepONet, FNO, transformers) all require a predefined domain; FBNO is the first to handle domains that evolve in time and are unknown before solving — a limitation that previously excluded an entire class of physically important problems.
  • Provides theoretical guarantees. A formal Universal Approximation Theorem tailored to free boundary problems underpins FBNO, ensuring the mathematical construction converges with arbitrary accuracy — not just empirical evidence.
  • Delivers massive speed-ups for multi-query settings. Once trained, FBNO processes tumour growth trajectories at 85 cases/second vs. 1.6×10-4 cases/second for conventional finite-element solvers — a 104-fold speed-up, making real-time clinical applications feasible.
  • Covers heterogeneous physics. Validated across three structurally different problems — the classical Stefan (phase transition), a thermal–structural coupling (multi-physics), and non-convex tumour growth — demonstrating broad applicability across the sciences.
  • Architecture-agnostic and training-paradigm-agnostic. FBNO integrates seamlessly with both physics-informed and purely data-driven training, and works with diverse neural operator architectures, lowering the bar for adoption in existing workflows.

Core Methods

  • Topological conjugacy framework: the free boundary problem is recast as a dynamical system with flow map Ft. A conjugate system over a fixed reference domain Ωref is constructed, and a homeomorphism H linking the two systems is learned, enabling predictions on evolving domains without needing prior geometric knowledge.
  • FBNO architecture (MIONet-based): two neural networks — Ĝ (conjugate phase field) and Ĥ (diffeomorphism mapping) — share a trunk network. The output χ of Ĥ is embedded in the branch of Ĝ via Hadamard product (element-wise multiplication), providing geometry-aware conditioning.
  • Diffeomorphic constraint: to guarantee invertibility, the spatial Jacobian determinant |JS| is constrained to remain strictly positive throughout training. This allows flexible solution spaces while preventing unphysical domain folding or tearing.
  • Physics-informed training: a composite loss combines initial-condition loss, fixed-boundary loss, free-boundary residual loss, PDE interior loss, and diffeomorphism regularization. Automatic differentiation through the architecture enables enforcing PDE constraints on the original domain via the chain rule.
  • Data-driven training: purely supervised learning on pairs of input functions and FEM ground-truth solutions. An autoencoder-based bijectivity regularization ‖Ĥ(·) ∘ Ĥ(·)-1 - I‖ ensures the learned diffeomorphism is invertible in the data-driven setting.
  • Adaptive initial domain handling: FBNO parametrizes variable initial domain geometries Ω0 via sensors distributed over a bounding superset, allowing the framework to handle problems where initial domain shapes themselves vary across examples.
  • Universal Approximation Theorem for FBPs: the paper proves that under mild assumptions (Banach function spaces with Schauder bases, compact sets, existence of a diffeomorphism to a reference domain), the composed operator F̂ approximates the true FBP solution operator to arbitrary accuracy.

① Extend FBNO to learn transferable operators across families of PDE types

Extending this paperFBNO is validated on three problems but must be retrained from scratch for each new PDE family. The paper's framework — built on MIONet — is architecture-agnostic, yet it does not exploit transfer across problem types. A direct extension: use deep transfer operator learning (conditional shift adaptation) so that a model pre-trained on Stefan problem variants can be fine-tuned with only a handful of samples for a new free boundary problem (e.g., plasma or mushy-zone solidification), drastically reducing the data generation cost that the paper itself identifies as a limitation.

Added value to my researchTransfer learning in PDE operators is analogous to transfer learning for geospatial prediction tasks — a core challenge in my spatiotemporal modeling work. The conditional-shift framework directly informs how I could adapt a physics-informed spatial model trained on one city's service accessibility dynamics to another city with very limited local data, addressing the scarce-label regime that limits geospatial foundation models.

Extending FBNO by coupling its diffeomorphism operator with a generative uncertainty-aware downscaling model would allow high-resolution, real-time prediction of these evolving features directly from coarse-resolution ESM outputs, sidestepping expensive numerical weather simulations.
