Promising directions of machine learning for partial differential equations

Steven L. Brunton · J. Nathan Kutz Published 2024-06-28 DOI 10.1038/s43588-024-00643-2 Source nature.com ↗
Core Conclusion

Machine learning is advancing PDE research along three mutually reinforcing fronts: discovering interpretable governing equations and coarse-grained closures from data (via sparse regression/SINDy/PDE-FIND), learning effective coordinate systems and reduced-order models that linearize or compress nonlinear dynamics (Koopman operators, autoencoders, POD-SINDy), and learning solution operators and accelerating numerical schemes (neural operators, DeepONet, ML-corrected solvers achieving ~100× speedup). Each front retains the classical virtues of interpretability and generalizability while dramatically lowering the cost of working with PDEs across science and engineering.

Core Methods

  • Sparse identification of nonlinear dynamics (SINDy / PDE-FIND): constructs a library of candidate PDE terms Θ and identifies a sparse coefficient vector ξ via sparse regression, recovering interpretable governing equations (e.g., Navier–Stokes Reynolds number to within 1%).
  • Weak-form PDE-FIND: integrates candidate terms over random control volumes to dramatically improve noise robustness — a major advance over the original PDE-FIND that enables application to experimental rather than only simulated data.
  • Symbolic regression and evolutionary algorithms (DL-PDE, DLGA-PDE, SGA-PDE): learn more general PDE structures combining deep-learning-generated data with genetic/evolutionary search, achieving greater expressivity than fixed libraries.
  • Koopman operator approximation: represents nonlinear PDE dynamics as linear evolution on a lifted latent space; approximated using dynamic mode decomposition (DMD) and deep autoencoder networks (encoder φe, linear operator K, decoder φd).
  • POD-Galerkin reduced-order models extended with ML: classical POD low-rank subspace combined with SINDy (Galerkin regression) or DMD to learn ROM dynamics in a data-driven rather than projection-based way; autoencoders generalize linear subspaces to nonlinear manifolds.
  • Neural operators (Fourier Neural Operator, multipole neural operator): parameterize the solution operator via integral kernel layers; enable discretization-flexible, mesh-free PDE solvers trained on high-fidelity data and evaluated at any grid location.
  • DeepONet: learns nonlinear operators via two co-trained networks (branch encodes the input function at sensor locations; trunk encodes query coordinates) — grounded in Chen & Chen's universal approximation theorem for operators.
  • ML-accelerated numerical solvers: learn improved finite-difference stencils, pre-conditioners, and coarse-grid correction terms (e.g., Kochkov et al. achieving ~100× speedup for 2D Kolmogorov flow on a grid 10× coarser per dimension).
