Unsupervised learning of topological non-Abelian braiding in non-Hermitian bands

Yang Long · Haoran Xue · Baile Zhang Published 2024-07-12 DOI 10.1038/s42256-024-00871-1 Source nature.com ↗
Core Conclusion

A novel unsupervised machine learning algorithm successfully identifies topological non-Abelian braiding in complex-energy (non-Hermitian) bands without requiring any pre-defined topological invariant. The algorithm's results are consistent with Artin's braid equivalence conditions, and the findings inspire a new quantized winding matrix as a universal topological invariant. The approach also reveals bulk-edge correspondence in non-Hermitian systems and can solve the unknotting problem — all in an unsupervised, label-free manner — opening new routes for topological quantum computing applications.

Core Methods

  • Similarity function construction: for each pair of non-Hermitian Hamiltonians Hi and Hj, a kernel-based similarity Kij is computed from the product of flattened-Hamiltonian determinants across all pairs (m,n) and all k-points in the Brillouin zone. Random chiral-symmetry-preserving perturbations are added to improve numerical robustness.
  • Unsupervised clustering algorithm: an incremental set-based clustering procedure (adapted from prior work by the same group) maintains a set S of representative samples; each new sample is compared against all S members and either merged with a matching cluster or added as a new phase. Requires no labels, no invariants, and no prior knowledge of the number of phases.
  • Winding matrix derivation: after clustering reveals topological equivalence, the quantized winding matrix wmn = (1/2πi)∮ ∂k ln[Em(k)−En(k)] dk is introduced as a closed-form topological invariant. Non-commutativity of w for N > 2 encodes the non-Abelian character.
  • Bulk-edge correspondence: the net number of skin modes Ns = Nl − Nr is linked analytically to ∑m,n wmn, establishing a non-Hermitian bulk-edge correspondence from the winding matrix.
  • Extension to 2D/3D exceptional semimetals: the 1D braiding topology is mapped to 2D/3D gapless systems via coordinate substitution (cos k → kx, sin k → ky), and the same algorithm is applied on the enclosing manifold S1.
  • Unknotting problem: the algorithm determines whether an arbitrary braid (closed via Brillouin zone periodicity) is topologically equivalent to an unknot by computing its similarity to a pre-generated set of known unknots.

① Replace the incremental clustering with a learned similarity metric for scalable phase discovery

Extending this paperThe paper's unsupervised clustering relies on a hand-crafted kernel similarity Kij derived from flattened Hamiltonians. This kernel is exact but scales as O(N2) in the number of Hamiltonian pairs and requires re-computation for every new sample. A direct extension: replace the fixed kernel with a contrastive or metric-learning network trained on the paper's own synthetic braid data (using the clustering assignments as weak supervision). Such a network would generalize to higher braid groups (B7, B8, …) and longer braid words without exhaustive recomputation of winding integrals, substantially scaling the framework beyond the B6 demonstration.

② Use the winding matrix as a training signal for supervised topological band-structure prediction

Extending this paperThe paper introduces the winding matrix w as a post-hoc theoretical invariant inspired by the clustering results. A forward extension: treat w as a ground-truth label and train a neural network that maps raw complex-energy band data (Brillouin zone spectra) directly to the winding matrix entries, bypassing the iterative clustering entirely for inference. The existing algorithm would generate large labeled corpora for B2–B6, and the network would be tested on out-of-distribution higher braid groups. This mirrors how ML was used to find band-structure-to-property mappings in materials science.

Added value to my researchThe same idea of learning invariant quantities from structured data as regression targets applies to my explainable ML for spatial prediction: local spatial statistics (local Moran's I, MGWR coefficients) can be treated as invariants to be predicted from feature maps, turning a computationally expensive spatial regression into a fast forward-pass inference at scale.

④ Extend the unknotting algorithm to topological data analysis of spatial networks

Extending this paperThe paper demonstrates that the algorithm can determine whether a closed braid is an unknot by comparing it to a reference set of known unknots via the similarity function. A mathematical extension: this is structurally identical to persistent homology-based classification of loops in a topological space. The paper's similarity function could be reformulated as a kernel for topological data analysis (TDA), enabling unsupervised detection of non-trivial loops and braids in general network data — extending from the Brillouin zone to arbitrary periodic graphs.

⑤ Port the bulk-edge correspondence to graph neural network expressivity analysis

Extending this paperThe paper's winding matrix establishes a bulk-edge correspondence: the number of skin modes localized at a boundary is computed from ∑m,n wmn, a bulk invariant. This is a non-Hermitian analog of topological protection: edge features are determined by bulk topology. A conceptual extension: map this to graph neural network (GNN) architecture design, where the "bulk" is the GNN message-passing topology and the "edge" modes are boundary node representations. Choosing graph architectures whose message-passing graph has a prescribed winding matrix could guarantee certain boundary expressivity properties — a physics-inspired approach to GNN design.

Added value to my researchMy physics-informed graph learning for spatiotemporal prediction could benefit from this topological perspective: choosing spatial graph connectivity such that the resulting graph has topological invariants aligned with known boundary effects (e.g., edge-of-city deprivation patterns or border effects in accessibility) would give the GNN inductive bias that currently requires manual feature engineering or large training datasets.
