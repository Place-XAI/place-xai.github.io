A large-scale unified deep learning model for peptide mass spectrum interpretation trained on multimodal data

Jiale Zhao · Pengzhi Mao · Kaifei Wang · Yiming Li · Yaping Peng · Ranfei Chen · Shuqi Lu · Xiaohong Ji · Jiaxiang Ding · Xin Zhang · Yucheng Liao · Weinan E · Han Wen · Weijie Zhang · Hao Chi Published 2026-05-25 DOI 10.1038/s42256-026-01234-8 Source nature.com ↗
Core Conclusion

pUniFind is a large-scale multimodal foundational model for proteomics that unifies open peptide-spectrum scoring and zero-shot de novo sequencing in a single framework trained on over 100 million spectra. It achieves a 42.6% increase in identified peptides in immunopeptidomics, identifies 60% more peptide-spectrum matches under modification-rich de novo conditions despite a 300× larger search space, and recovers an additional 38.5% of peptides in regular de novo sequencing — including 1,891 genome-mapping peptides absent from reference proteomes. A quality control module further raises consistency with RNA-Seq evidence from 65.4% to 85.0%.

Why It Matters

Prior deep learning tools in proteomics were either re-scoring engines or de novo sequencers; pUniFind is the first to integrate both into one cross-modal pretraining framework, removing the traditional boundary between database search and de novo paradigms.
  • Dramatically expands the dark proteome. By recovering 1,891 genome-mapped peptides absent from reference proteomes and improving RNA-Seq consistency from 65.4% to 85.0%, pUniFind unlocks biology invisible to conventional database search — including novel immunotherapy targets in immunopeptidomics.
  • Orders-of-magnitude larger modification search space without accuracy loss. The modification-rich de novo workflow operates in a space 300× larger than existing methods while still identifying 60% more peptide-spectrum matches, setting a new benchmark for open modification discovery.
  • Practical and reproducible infrastructure. The model ships as a Windows GUI, Linux CLI, web server, and Code Ocean capsule, making it accessible without specialized hardware — lowering the barrier for clinical and translational proteomics labs worldwide.

Core Methods

  • Cross-modal pretraining: pUniFind aligns two heterogeneous modalities — tandem mass spectra and peptide sequences — via cross-modality prediction pretraining tasks, enabling both open end-to-end scoring and zero-shot de novo sequencing within a single shared architecture.
  • Open scoring (end-to-end peptide-spectrum scoring): rather than closed-database re-scoring, pUniFind performs open scoring across a modification-expanded peptide space without pre-specifying which PTMs exist, dramatically improving sensitivity in immunopeptidomics and modification-rich datasets.
  • Two de novo sequencing workflows: (1) Modification-rich de novo — exploits open scoring to search a 300× larger modification space than existing de novo methods; (2) Regular de novo — recovers peptides with full fragment ion coverage and high consistency with database-search results, including 1,891 genome-mapping peptides absent from reference proteomes.
  • Quality control module: deep-learning-derived features from the shared encoder are used to filter and rank de novo outputs, validated against RNA-Seq transcriptomic evidence, raising consistency from 65.4% to 85.0%.
  • AlphaFold 3 structural validation: identified HLA-peptide complexes from immunopeptidomics analysis are structurally predicted via AlphaFold 3 (Supplementary Figs. 17–34) to validate the biological plausibility of novel identifications.
  • Benchmarking: compared against traditional search engines (MSFragger, MaxQuant, pFind) and de novo tools (pNovo, Casanovo, InstaNovo) across TIMS/timsTOF and Astral/Orbitrap instrument platforms.

② Extend QC-by-RNA-Seq into a cross-domain orthogonal-evidence supervision signal

Extending this paperpUniFind's quality control module treats RNA-Seq consistency as a post-hoc filter. A deeper extension of this paper would incorporate RNA-Seq consistency as a soft training signal during fine-tuning — a differentiable cross-evidence loss that penalizes de novo results inconsistent with transcriptomics. This requires no new data collection: the 27 PXD training datasets in PRIDE already include projects with paired RNA-Seq experiments, enabling a fully self-supervised multi-evidence loop. Testing this on the existing evaluation benchmarks would quantify how much additional performance can be extracted from existing public data through smarter supervision design rather than more model parameters.

③ Extend open-modification scoring to glycopeptidomics as the next combinatorial frontier

Extending this paperpUniFind's open scoring abandons the fixed-modification-database assumption, operating successfully in a 300× larger search space. An immediate extension of this paper is glycopeptidomics: glycan modifications create a combinatorial space many orders of magnitude larger than standard PTMs, making them the hardest test case for open scoring. The paper already uses ion-mobility instruments (TIMS/timsTOF) that produce glycan-informative cross-section data; integrating ion-mobility drift time as an additional spectral feature channel into pUniFind's encoder would be a modular extension. The corpus contains exactly this counterpart — ion-mobility de novo glycan sequencing with a self-expanding database — providing a direct technical bridge to validate pUniFind's open-scoring framework on glycan combinatorics.

Added value to my researchConceptually, pUniFind's "open search" over an exponentially large combinatorial space without pre-specifying candidates is the proteomics analogue of my open accessibility measurement problem in 2SFCA/R2SFCA: defining service catchment areas without fixing a travel-time threshold a priori. This could unify 2SFCA variants (Gaussian, kernel-based, floating) into a single learned open-decay framework.

④ Apply quantization and scaling analysis to establish a pUniFind-lite for clinical deployment

Extending this paperpUniFind trained on 100M+ spectra but the paper does not report model parameter count, memory footprint, or inference latency — critical practical metrics for clinical adoption. A direct extension of this paper is to apply quantization (INT8/INT4) and knowledge distillation to produce a pUniFind-lite suitable for CPU-only clinical lab computers. The corpus contains exactly this analysis for a comparable-scale biology foundation model (Geneformer, 100M+ single-cell transcriptomes), demonstrating that quantization preserves predictive accuracy while drastically reducing memory. ⑤ Connect pUniFind's immunopeptidomics output to a multi-task neoantigen prioritization head

Extending this paperpUniFind achieves a 42.6% gain in immunopeptidomics peptide identification but its output is a list of ranked peptide identifications, not directly actionable immunotherapy candidates. A targeted extension of this paper would attach a multi-task prediction head to pUniFind's shared encoder that simultaneously predicts HLA-binding affinity, proteasomal cleavage probability, and T-cell recognition likelihood — converting raw identifications into prioritized neoantigen candidates. The corpus contains a multi-task transformer model that bridges exactly this gap (peptide presentation → T-cell recognition), trained on the same HLA-peptide binding data that pUniFind's de novo pipeline generates. Connecting the two pipelines would create an end-to-end closed-loop system from raw mass spectra to ranked immunotherapy targets, requiring no additional wetlab experiments.
