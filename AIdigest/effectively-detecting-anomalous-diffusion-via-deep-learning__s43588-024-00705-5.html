Effectively detecting anomalous diffusion via deep learning

Adrian Pacheco-Pozo · Diego Krapf Published 2024-10-11 DOI 10.1038/s43588-024-00705-5 Source nature.com ↗
Core Conclusion

A deep learning network trained on five canonical anomalous diffusion models — CTRW, ATTM, FBM, Lévy walks, and SBM — can classify single-particle tracking trajectories while simultaneously detecting whether a trajectory belongs to a model outside the training set. By incorporating out-of-distribution (OOD) outlier trajectories from five additional diffusion models into training, the classifier reliably flags unknown dynamics rather than forcing them into a known category, making it directly applicable to real biological and physical experiments where the true underlying process may be unanticipated.

Core Methods

  • Deep learning trajectory classifier: a neural network trained end-to-end to map raw SPT trajectories to one of five in-distribution anomalous diffusion models (CTRW, ATTM, FBM, Lévy walks, SBM) or to an out-of-distribution flag.
  • OOD-augmented training: five additional diffusion models (DBM, TSM, CBM, Sinai, OU) are injected as outlier classes during training, teaching the network to recognize the statistical signatures of unknown dynamics without treating them as in-distribution categories.
  • Open-set recognition: at inference time, trajectories are either assigned to a known in-distribution model or classified as OOD, producing reliable predictions on open-world experimental data.
  • End-to-end feature learning: the network learns discriminative representations directly from position time series, capturing memory, heavy tails, ageing, and confinement signatures without hand-crafted features such as MSD exponents.
  • Experimental validation: the trained classifier is applied to biologically obtained single-particle trajectories to demonstrate practical applicability and to identify when the physical process departs from all trained models.

① From OOD-augmented diffusion classification to multi-regime trajectory segmentation

Extending this paperThe paper classifies each trajectory as a whole into one in-distribution model or flags it as OOD. Real biological trajectories frequently exhibit switching behavior — a particle alternating between confined diffusion and free Brownian motion at different time windows. A direct extension: combine the paper's OOD-augmented training with a temporal change-point detection layer that segments each trajectory into sub-windows and assigns a model (or OOD flag) per segment. The companion paper (10.1038/s43588-024-00703-7) addresses OOD uncertainty differently; merging both architectures could yield a robust segmentation-plus-detection pipeline for heterogeneous biological data.

Added value to my researchMy GPS and mobile-phone trajectory analysis regularly encounters heterogeneous movement regimes — commuting, loitering, transit — where no single mobility model fits the whole trace. Adapting the OOD-augmented training paradigm to human mobility trajectories would identify anomalous behavioral patterns (unusual routing, sudden mode switches, mobility deprivation episodes) without forcing them into prespecified categories, extending my spatiotemporal kernel density and agent-based modeling toolkit with a data-driven anomaly flag.

② Replace the sequence head with a geometric deep learning architecture for multi-particle graph inputs

Extending this paperThe current network treats each trajectory as an independent 1D sequence. In real SPT experiments, many particles are tracked simultaneously and their spatial relationships — proximity, correlated motion, confinement by cell boundaries — carry additional information about the underlying diffusion mechanism. Extending this paper: represent the ensemble of co-tracked particles as a dynamic spatial graph and apply a geometric deep learning layer that propagates information across neighboring tracks. This would improve classification accuracy in crowded cellular environments and enable detection of collective anomalous diffusion regimes that single-track classifiers necessarily miss.

Added value to my researchThe geometric graph perspective maps onto my physics-informed graph learning agenda. In spatial accessibility modeling, facilities and demand points form a bipartite graph where learned message-passing can encode geometric constraints and OOD pattern detection simultaneously — for instance, flagging a healthcare facility catchment whose actual usage pattern deviates from all modeled service-area geometries, signaling an unmet-need anomaly.

③ Open the black box: attribution methods to extract interpretable physical features

Extending this paperThe paper demonstrates reliable classification and OOD detection but does not open the black box to reveal which trajectory statistics — autocorrelation structure, displacement distribution tails, ergodicity breaking — drive each prediction. Extending it: apply attribution methods (SHAP, integrated gradients, or attention rollout over the sequence) to identify discriminative features, then verify whether they match known analytical fingerprints of each diffusion model. This yields a learned biophysical feature dictionary that simultaneously validates the network, deepens physical understanding, and produces human-readable diagnostics alongside the classification output.

④ Extend the model set to include non-equilibrium and actively driven diffusion in biological media

Extending this paperThe paper trains on ten diffusion models drawn from the AnDi challenge set, which is comprehensive but not exhaustive. Missing are regimes particularly relevant to membrane biophysics: hop diffusion across domain boundaries, transient binding kinetics (adsorption–desorption superimposed on free diffusion), and active-transport components (motor-driven ballistic phases). Extending the paper: add these as additional in-distribution or outlier model classes using the same OOD-augmented framework — the architecture is already designed to accept new model entries. The companion sensor work on particle mobility at single-molecule resolution provides concrete experimental contexts for these extended model classes.

⑤ Formalize the OOD-augmentation principle as a benchmark construction standard for scientific ML

Extending this paperThe paper's methodological core — deliberately including out-of-distribution outlier classes during training to improve reliability on unknown inputs — is applied to biophysics but is framed as a general principle. A generalizing extension: formalize this as a benchmark construction methodology for scientific machine learning. For any physical classification task (fluid flow regime, material phase, reaction pathway), the benchmark should include both a target model set and a curated OOD perturbation set; the classifier's reliability on held-out OOD categories becomes a standard evaluation metric alongside in-distribution accuracy. This could be as influential for ML on physics data as the AnDi challenge itself was for anomalous diffusion.
